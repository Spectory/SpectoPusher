Example of a secure connection to Coyote socket.

Requirements:

  1. Make sure Coyote server is running with `UNSECURED_SOCKET` env var off (see `dev.exs` file).
  2. run `iex -S mix` from the server folder, and use `Coyote.Auth.sign_all [1,2]` to generate 2 tokens.
  3. Open this page in 2 browsers, enter the generated tokens and click connect
(See method Coyote.Auth.sign on Coyote server)
